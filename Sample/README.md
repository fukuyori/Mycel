# 業務改革マインドマップ サンプル

目的は、業務改革を「現状把握 → 課題構造 → 施策 → 効果測定 → 定着」まで一枚の地図として整理すること。

Mycel でこの `Sample` フォルダを開くと、改革プロジェクトの全体像をマインドマップとして確認できます。

## 見方

- `01_現状把握`: いま何が起きているか
- `02_課題構造`: 問題のつながり
- `03_改革テーマ`: 重点テーマ
- `04_施策アイデア`: 実行候補
- `05_KPIと効果`: 測定方法
- `06_ロードマップ`: 進め方
- `07_コミュニケーション`: 関係者との合意形成
- `08_リスク管理`: 失敗要因と対策
