# Changelog

Mycel の主な変更履歴です。

## 0.2.7

- ファイルを別ファイルの右端へドロップすると、実ファイルを移動せずに関連線を作成して `.mycel/links.json` に保存する機能を追加
- 関連先ファイルをリンク元ファイルの右側へ配置し、プレビュー表示時も重なりにくいように調整
- 関連線を点線ではなく通常の実線で表示し、関連先ファイルは元のフォルダ接続線を非表示にするように変更
- ファイルのコンテキストメニューから関連を解除し、元のフォルダ接続へ戻せるように追加

## 0.2.6

- 編集ペインを左、右、下に配置できるメニューを追加し、選択した位置を次回起動後も維持するように変更
- キャンバスの拡大率、表示位置、ウィンドウサイズ、最大化/全画面状態を `.mycel/view.json` に保存し、次回起動時に復元するように変更
- 起動時に Editor が ON の場合、編集ペインの表示状態を確実に同期するように修正
- 編集ペイン内の `Ctrl + S` で保存し、編集中ファイルへフォーカスを戻すように追加
- 編集ペイン内の `Ctrl + S` で、1行目の `Subject: filename.txt` に合わせてファイル名を変更し、その `Subject:` 行を本文から削除できるように追加
- `Subject:` / `Key:` で始まる行をインラインプレビューに表示しないように変更

## 0.2.5

- ルート配下のファイル数が多い場合、起動時はルート直下のフォルダまで表示し、それ以下を折りたたむように変更
- フォルダ選択時の `N` / `Shift + N` で `NewFile.txt` / `NewFolder` を作成するショートカットを追加
- ファイル選択時の `N` で、そのファイルと同じフォルダに `NewFile.txt` を作成するように変更
- `Ctrl + C` / `Ctrl + V` で選択ファイルやフォルダを同じ親フォルダへコピーできるように追加
- `E` で選択ファイルを右側編集ペインへフォーカスできるように追加
- `V` で選択ファイルのプレビューを開閉できるように追加し、複数選択時は全て開いていれば閉じ、1つでも閉じていれば開くように変更
- `Tab` / `Shift + Tab` で表示中の同階層項目を前後に移動できるように追加
- `N`、`Shift + N`、`V`、`Tab` 操作後にキャンバスのフォーカスと選択状態を維持するように修正
- 右側編集ペインで編集中のファイルがプレビュー表示中の場合、自動保存後にプレビューを更新するように変更
- 複数選択したファイルとフォルダのドラッグ、フォルダへのドロップ、削除の動作を改善
- ドラッグ中にドロップ先フォルダをハイライトするように追加

## 0.2.4

- 単一ファイルのコンテキストメニューにプレビューを開く/閉じる操作を追加
- プレビューを開く/閉じる操作をコンテキストメニュー上部へ移動
- 色調整ダイアログを廃止し、コンテキストメニューの固定色パレットから色を選べるように変更
- 色表示を接続線ではなく、ファイル名、アイコン、プレビュー領域の淡い背景色として表示するように変更
- 色メニューの表示を色付き丸アイコンと色名に整理
- インラインプレビュー内の文字を選択してコピーできるように変更
- プレビュー本文のコンテキストメニューに「コピー」を追加
- キャンバスのズームを、Ctrl なしのマウスホイールで行うように変更
- `--version` でバージョンを表示して終了する機能と、タイトルバーのバージョン表示を追加
- 選択した編集可能テキストファイルを右側で直接編集し、自動保存する編集ペインを追加
- エディタ内の Ctrl + マウスホイールで変更した文字サイズを、他のファイルや次回セッションでも維持するように変更
- ツールバーの Editor ボタンと Ctrl + E で、右側編集ペインの表示/非表示を切り替えられるように変更

## 0.2.3

- ファイルのコンテキストメニューから開けるテキストファイル用の内蔵エディタを追加
- 内蔵エディタに Ctrl + S 保存、未保存変更の確認、外部変更時の上書き確認を追加
- 内蔵エディタで Ctrl + マウスホイールによる拡大/縮小に対応
- 仮想ホワイトボードキャンバスを広げ、ノードの外側へ自由に表示位置を移動できるように変更

## 0.2.2

- 選択されたファイルとフォルダの表示を統一し、アイコンと名前の範囲を枠と塗りで強調するように変更
- 通常状態のフォルダとルートフォルダの枠をなくし、選択中の項目とドロップ対象だけが目立つ表示に変更
- `.mycel/previews.json` にインラインプレビューの表示/非表示、幅、高さを保存
- 起動時、フォルダを開いた時、全体リロード時に保存済みのプレビュー状態を復元
- 何もない場所での右ボタンドラッグによるキャンバス移動を追加

## 0.2.1

- Ctrl + 左クリックによる複数選択を追加
- Shift + フォルダダブルクリックで、ダイアログなしに `NewFolder` を作成
- ファイルまたはフォルダを単一選択して F2 でインライン名前変更
- F5 の全体リロード、F11 の最大化/通常表示、`?` のチートシート表示を追加
- 起動時に、引数なしならカレントディレクトリ、引数ありなら指定ディレクトリを開くように修正
